CountEx-VQA:
Counterfactual Explanations for Visual Question Answering

CountEx-VQA is a tool that generates counterfactual images to make a Visual Question Answering (VQA) model's more interpretable. Given an image-question pair and a VQA model, CountEx-VQA is trained to apply the minimal number of modifications to the image in order to change the VQA model's predicted answer to the question. This demo uses a pre-trained MUTAN model

How it works

  1. Upload an image and enter a question in the form below.
  2. Hit the "Submit" button
  3. CountEx-VQA uses a pretrained MUTAN VQA model to predict an answer to your question. In addition to the prediction, it outputs a heatmap that indicates on which regions of the image MUTAN focused the most to make its decision.
  4. CountEx-VQA returns a counterfactual image along with MUTAN's new prediction and the corresponding heatmap.

This demo is part of the Master's Thesis Generating Counterfactual Images for VQA by Editing Question Critical Objects, pursued by Timo Hartmann at WeST, University Koblenz-Landau.
